#### 1. 이해

- 가능한 모든 경우의 수를 구하는 문제이므로 백트래킹 알고리즘을 사용하는 것이 좋다.

#### 2. 계획

- 이 문제는 순열 중에서 특정 조건의 순열만 남기는 문제이다.

#### 3. 실행

#### 4. 반성

- 백트래킹 문제에서는 솔루션에 해당할 때, 조건, 그리고 가능한 경우의 수를 고려하여 알고리즘에 적용하자.
- GOAL에 해당할 때 반환하는 값은 실제로 backtrack에서 반환하는 값과 동일한 형태를 띄고 있어야 한다.
